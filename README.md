# EventSauceBundle example app

Clone project

```bash
git clone https://github.com/andrew-pakula/eventsaucebundle-example.git
```

Run inside project

```bash
make up # builds dockers
make setup # installs dependencies, runs migrations
```
Open

http://localhost:8888/docs \
root@localhost:3307
